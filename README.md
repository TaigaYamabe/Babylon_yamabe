Babylon_yamabe.php
ファイルのアップロード機能をつけてサーバー側の処理を行うため、拡張子が.phpになっています。
phpの実行環境はMAMPを使用して行っています。

GLB形式のデータはglb形式サンプルデータのフォルダの中にいくつか入っているためアップロードを試しに行ってみてください。

カメラ移動はマウスと十字キー

オブジェクトの移動はマウスのクリック選択で動かすオブジェクト選択後
左右前後：WASDキー
上下：QEキー
回転：1,2キー
拡大縮小：3,4キー
削除：5キー
